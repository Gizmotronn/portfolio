About P.folio

P.folio is a powerful Responsive Portfolio HTML5/CSS3 template based on Bootstrap 4. P.folio is Fully Responsive and Compatible with all types of Screens, Devices and Bowsers. Very simple and clean design anyone can use this template and customize as his/her wishes.Anyone can use this any personal area like Personal resume, cv, Web design, photography, business, doctor, teacher and many more. It’s very easy to customize and also well documented so it’ll suit your needs. If you would use our template P.folio, obviously you will be attracted.

  2.  <div class="container">
  3.   <div class="row">
  4.    <div class="col-md-12">
  5.     <nav class="navbar navbar-expand-lg navbar-light">
  6.      <button class="navbar-toggler">
  7.       <span class="navbar-toggler-icon"></span>
  8.      </button>
  9.      <a class="navbar-brand" href="#"><img src="images/logo.png" alt=""></a>

  11.     </div> 
  12.    </div>
