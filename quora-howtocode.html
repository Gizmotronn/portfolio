How do I learn a coding language?

by Liam Arbuckle

Original Question: I want to learn computer programming for free. Who can help me? There are always people that are able to help you learn programming. However, you need to carefully pick a language that you want to learn. There are many different languages you can choose from, here are a few good ones: Python. Python is a very easy-to-learn language that is used in many applications, from data science to web programming. Many job agencies require knowledge of Python. I’d definitely recommend learning Python first. HTML. HTML may seem hard to learn at first, but if you make use of a good code editor it’s not too hard. I’d recommend learning CSS at the same time. Both are used for website design and are actively maintained. C#. C#, like Python, is used in many areas but is very good for game design, an example of which would be Unity. Now you need to figure out how you’re going to learn your chosen language. I’ll assume we’re using Python, however any language will work. I’d recommend watching video lectures, like the ones on Udemy, as they are usually good quality and include exercises that help you practice what you’ve just learnt. While most courses on Udemy are paid, there is a good amount that are free. Also check out youtube as well for specific tutorials
